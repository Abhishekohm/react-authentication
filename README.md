# REACT-AUTHENTICATION

## Features

- ### Login
- ### Logout
- ### Persistent Login
- ### Authorization using JWT
- ### Dynamic creation of access token
- ### API requests to backend using axios and preloaded access token

## Tech Stack

- ### Bcrypt
- ### JWT
- ### MongoDB
- ### Axios
